 
1.Python program to create a dictionary with integer keys, and print the keys, values & key-value pairs 

# Create dictionary with integer keys
my_dict = {1: "Adhul", 2: "Manoj", 3: "Sarmila"}

# Print keys
print("Keys:", my_dict.keys())

# Print values
print("Values:", my_dict.values())

# Print key-value pairs
print("Items:", my_dict.items())

OUTPUT:
Keys: dict_keys([1, 2, 3])
Values: dict_values(['Adhul', 'Manoj', 'Sarmila'])
Items: dict_items([(1, 'Adhul'), (2, 'Manoj'), (3, 'Sarmila')])


2.Python program to create a dictionary with mixed keys, and print the keys, values & key-value pairs

# Create dictionary with mixed keys
my_dict = {1: "A", "B": 2, 3: "C"}

print("Keys:", my_dict.keys())
print("Values:", my_dict.values())
print("Items:", my_dict.items())

OUTPUT
Keys: dict_keys([1, 'B', 3])
Values: dict_values(['A', 2, 'C'])
Items: dict_items([(1, 'A'), ('B', 2), (3, 'C')])

 1.Python Program for Adding, removing elements in the list.

State = ["Tamil Nadu", "Kerala"]

# Add
State.append("Goa")
State.insert(1, "Karnataka")

# Remove
State.remove("Kerala")
popped_item = State.pop()  # Removes last item

print("States:", State)
print("Popped Item:", popped_item)

OUTPUT:
States: ['Tamil Nadu', 'Karnataka']
Popped Item: Goa

 3. Python Program to print a two list using single ‘FOR and IN’ loop. 

list1 = [1, 2, 3]
list2 = ["A", "B", "C"]

for x, y in zip(list1, list2):
    print(x, y)

OUTPUT:

1 A
2 B
3 C

4.Python Program to input, append and print the list elements. 

my_list = []

n = int(input("Enter number of elements: "))
for _ in range(n):
    item = input("Enter element: ")
    my_list.append(item)

print("Final List:", my_list)

OUTPUT:

Enter number of elements: 3
Enter element: apple
Enter element: banana
Enter element: mango
Final List: ['apple', 'banana', 'mango']
